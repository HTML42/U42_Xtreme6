# user.class.md

## Rolle
Singular-Objekt `User` für eine einzelne User-Instanz.

## Anforderungen
- Speichert `id`, `name`, `email`.
- Kann als Domain-Objekt in PHP und JavaScript verwendet werden.
- Validiert minimale Eingaben (`name` und `email` nicht leer).

## KI-Output-Ziele
Aus dieser Datei sollen generiert werden:
- `user.class.php`
- `user.class.js`
- `user.test.php`
- `user.test.js`
